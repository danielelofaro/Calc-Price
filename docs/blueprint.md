# **App Name**: CalcoloPrezzi Pro

## Core Features:

- Base Price Input: Allow users to input up to three individual starting prices that sum to the total base price.
- Discount Application: Provide fields to apply up to three percentage-based discounts to the base price.
- Markup Application: Provide fields to apply up to three percentage-based markups to the base price.
- Real-time Calculation Display: Instantly display the final calculated price as inputs are changed, reflecting all applied discounts and markups.
- Clear Input Management: Enable users to easily add or remove additional price, discount, or markup fields as needed.

## Style Guidelines:

- Primary color: A sophisticated, muted blue (#335699) conveying precision and reliability. This provides a professional anchor for interactive elements and text on a light background.
- Background color: A very subtle, cool-toned off-white (#F6F7F8) providing a clean, elegant canvas that is visually harmonious with the primary color, contributing to a modern aesthetic.
- Accent color: A vibrant light cyan (#52CAE0) for highlights, calls-to-action, and interactive elements. Its brightness and hue differentiation from the primary color ensure excellent contrast and draw attention where needed, offering a touch of modernity.
- Headline and body font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring high readability for numerical data and input labels, aligning with an elegant and functional interface.
- Use minimalist, outline-style icons for adding/removing fields and other functionalities to maintain a clean and elegant visual consistency.
- Implement a clean, organized, and responsive single-page layout. Input fields for base prices, discounts, and markups should be logically grouped in distinct sections for clarity, allowing for a focused user experience on any device.
- Incorporate subtle, swift animations for adding/removing input fields or confirming calculations, enhancing user interaction without distracting from the primary task.
